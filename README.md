# Hytale UI Plugin

IntelliJ IDEA plugin for Hytale UI file support.

<!-- Plugin description -->
Provides comprehensive support for Hytale UI (.ui) files in IntelliJ-based IDEs.

## Features
- **Syntax Highlighting:** Color-coded keywords, properties, strings, numbers, and comments
- **Code Completion:** Auto-complete for UI components, properties, and values
- **Code Formatting:** Automatic indentation and code formatting
- **Error Detection:** Real-time validation and error highlighting (Work in progress)
- **Auto-closing:** Automatic closing of brackets, braces, and quotes

## Coming Soon
- **Reference Resolution:** Navigate between imports and variable declarations
<!-- Plugin description end -->

## Installation

1. Download the plugin from the JetBrains Marketplace
2. Install via `Settings > Plugins > Install from disk...`
3. Restart your IDE

## Usage

Simply open any `.ui` file and the plugin will automatically provide syntax highlighting, code completion, and formatting support.
